Funcao PraqueServe Exemplo
alert Mostra um alerta na tela alert("mensagem de alerta");
console.log Mostra no console do browser console.log("mostra mensagem no console, bom para debug");
toLowerCase() Transforma toda a string maiuscula para minuscula string.toLowerCase();
toUpperCase() Transforma toda a string minuscula para maiuscula string.toUpperCase();
Arrays Cria uma lista de elementos que podem ser iguais ou diferentes em seus tipos (string, numero, booleano,funcoes,etc...) var lista = ["uva","pera"];
Adicionar elemento no Array Adiciona um elemento na posicao final do array lista lista.push("abacate");
Remove elemento do Array Remove um elemento da primeira posicao do array lista lista.pop();
length Mostra o tamanho da lista, normalmente a quantidade de elementos da mesma. lista.length;
reverse Faz a lista ficar ao reverso. lista.reverse();
toString Transforma uma lista em uma string. lista.toString();
